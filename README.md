
# StepLock

StepLock is a productivity Android app that helps users control screen time using **time-based series app locking**.

## Features

- Create multiple **series schedules**
- Select **allowed apps** with target usage time
- Define **blocked apps**
- **Weekly day selection**
- **Start and end time window**
- **Daily reset**
- Prevent conflicts between allowed and blocked apps

## Project Structure

- Kotlin
- Android SDK
- Gradle build system

This project is a starter implementation based on the design concept.
